Detailed Description

A single point in a sketch.

IMPORTANT: in order to mantain resolution independence and still have values that make sense, x and y are normalized values in [0,1] relative to the image size, while variance is a normalized value in [0,1] that maps, in pixels, to the [MIN_POINT_SIZE,MAX_POINT_SIZE] when considering the original resolution image.

This means that if you scale the original resolution of, e.g. 5184 × 3456 px to 1500 x 1000 px, a point with x=0.5, y=0.5, and variance=0.5 will indicate a circle centered in (2592, 1728) with a size of 90.5 pixels in the original image but since the rescaled image is only ~28,93% of the original size, the same values will indicate a circle centered in (750, 500) with a size of 26.19 pixels
